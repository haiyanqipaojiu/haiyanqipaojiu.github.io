1.html的所有东西 都称为标签

1.1 标签以两种形式编写

1.2 一个标准的html文档结构如下

：告诉浏览器以下为html文档

标签html ：文件本身

标签head ：文件头部(前缀)

标签body ：文件身体(内容)

2. div标签

特性是一行一行 绝大多数网页所使用的制作的“砖块”

3. img 标签

网页就是由 文字 + 图片 + 媒体（音频 + 视频）

img 标签 在网页可以让图片显示出来

<img> 不需要闭合

需要注意的是绝对地址与相对地址的问题

4. a 跳转标签

跳转标签<a></a>
href=”里面放入要跳转的地址”

target=’_blank’ ： 原页面不跳转 打开一个新页面

如果想点击图片进行跳转
就是a标签里面放入img标签即可

1. 列表 ol/ul

1.1 无序列表

正规的写法 ul下面的第一层只能放li [W3C标准]

1.2 有序列表

有序列表

ol

ol和ul的用法基本一致

ol下面的 第一层 最好是 li [W3C]

只不过ol的下面 是默认使用数字标注 1 ～ n

2. 段落标签 p

2.1

p 标签

一般是段落 或者 新闻 或者特别多的文字 去使用的标签

默认是一行的

而且默认每行的上下的留白会有 比起其他的元素。

3. input 标签

input插件的东西

type=’input的类型’

text 默认

password 密码

radio 单选

checkbox 复选

color 颜色

button 按钮

4. 按钮标签 input button

4.1 input按钮

input

type=’button’

value=’内容’

4.2 button按钮
