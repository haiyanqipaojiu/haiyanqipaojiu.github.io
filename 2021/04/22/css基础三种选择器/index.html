2.3 类选择器

class 用的比较广范 比较多

id>class>标签选择器

3.进阶选择器

3.1 后代选择器

后代选择器 更精准的选择需要的元素

语法: 父标签 子标签{}

示例 : .myDiv .lastNode p{background: green;}

这里的后代指的是 层次中的代级 .#等没有优先级

可以有无限多层后代
